This very site! Built as a static, bilingual portfolio using Astro and Tailwind CSS, deployed to Cloudflare Pages.

## Tech Stack

- **Astro** — Static site generation with island architecture
- **Tailwind CSS v4** — Utility-first styling
- **Cloudflare Pages** — Fast, globally distributed hosting
- **Content Collections** — Type-safe markdown content

## Goals

- Fast first contentful paint
- Bilingual support (EN/ES) out of the box
- Clean, minimal design
