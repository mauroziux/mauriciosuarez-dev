Una herramienta CLI minimal para definir y ejecutar tareas de desarrollo con un simple archivo de configuración YAML.

## Características

- Definiciones de tareas en un solo archivo `tasks.yml`
- Ejecución de tareas en paralelo y secuencial
- Modo watch integrado para desarrollo
- Extensible mediante plugins

## Por qué

Quería una alternativa más simple a herramientas de build complejas para la automatización cotidiana de proyectos.
