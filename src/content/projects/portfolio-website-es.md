¡Este mismo sitio! Construido como un portafolio estático bilingüe usando Astro y Tailwind CSS, desplegado en Cloudflare Pages.

## Stack Tecnológico

- **Astro** — Generación de sitios estáticos con arquitectura de islas
- **Tailwind CSS v4** — Estilos utility-first
- **Cloudflare Pages** — Hosting rápido y distribuido globalmente
- **Colecciones de Contenido** — Contenido markdown con tipos seguros

## Objetivos

- First contentful paint rápido
- Soporte bilingüe (EN/ES) desde el inicio
- Diseño limpio y minimalista
